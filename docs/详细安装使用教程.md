# cchat 详细安装与使用教程

> 当前版本已经同时支持两类本地会话：
>
> - Claude Code
> - Codex
>
> 以后你主要只需要记住一个命令：`cchat`。

---

## 1. 先看结论：现在应该怎么用

如果你只想记住最少的内容，只记下面这些就够了：

```bash
cchat
cchat "关键词"
cchat --list
cchat --list --date 2026-04-15
cchat --source claude --list
cchat --source codex --list
cchat --popup
cchat --popup "关键词"
```

这些命令分别表示：

- `cchat`：打开 macOS 弹窗界面
- `cchat "关键词"`：在终端里直接搜索本地会话
- `cchat --list`：列出最近会话
- `cchat --list --date 2026-04-15`：查看某一天的会话
- `cchat --source claude --list`：只看 Claude Code 会话
- `cchat --source codex --list`：只看 Codex 会话
- `cchat --popup`：强制用弹窗模式
- `cchat --popup "关键词"`：打开弹窗并预填关键词

默认情况下，`cchat` 会同时搜索：

- Claude Code 会话
- Codex 会话

---

## 2. `cchat` 和 `csearch` 现在分别是什么

### 2.1 `cchat` 是什么

`cchat` 是你日常唯一需要记住的主命令。

它现在统一负责：

- 搜索 Claude Code 本地会话
- 搜索 Codex 本地会话
- 在终端列出会话
- 在 macOS 弹窗中选择会话
- 生成正确的恢复命令
- 在 Terminal 中打开恢复命令

简单理解：

- `cchat` 是统一入口
- 你以后主要只围绕它建立使用习惯

---

### 2.2 `csearch` 是什么

`csearch` 仍然存在，但现在更像是 `cchat` 背后的内部搜索引擎。

它负责底层工作：

- 读取 Claude Code 的本地 JSONL 会话文件
- 读取 Codex 的本地 JSONL 会话文件
- 解析标题、时间、用户消息、助手消息
- 做关键词搜索、日期过滤、项目过滤
- 输出普通文本或 JSON 结构化结果

简单理解：

- `cchat`：面向日常使用
- `csearch`：面向底层实现、调试、复用

正常使用时，你只需要记住 `cchat`。

## 3. 现在支持哪些来源

## 3.1 Claude Code

默认读取目录：

```bash
~/.claude/projects
```

这个目录下通常会有很多 `.jsonl` 文件，每个文件对应一个 Claude Code 会话。

---

## 3.2 Codex

默认读取目录：

```bash
~/.codex/sessions
```

Codex 的目录结构一般是按日期组织，例如：

```bash
~/.codex/sessions/2026/04/15/rollout-xxxx.jsonl
```

每个 `.jsonl` 文件对应一条 Codex 会话。

---

## 3.3 默认行为

如果你不指定 `--source`，`cchat` / `csearch` 会默认同时搜索两个来源：

- Claude Code
- Codex

如果你只想看其中一个，可以使用：

```bash
cchat --source claude --list
cchat --source codex --list
```

## 4. 使用前必须先确认什么

正式使用前，建议先检查以下内容。

### 4.1 Python 3

```bash
python3 --version
```

如果能输出版本号，说明 Python 3 可用。

---

### 4.2 Claude Code / Codex 是否安装过

检查命令：

```bash
which claude || true
which codex || true
```

说明：

- 不要求你同时装两个客户端
- 但如果你要恢复某个来源的会话，对应命令最好可用

例如：

- 恢复 Claude Code 会话，需要 `claude`
- 恢复 Codex 会话，需要 `codex`

---

### 4.3 本地会话目录是否存在

先检查 Claude Code：

```bash
ls ~/.claude/projects 2>/dev/null || true
find ~/.claude/projects -name '*.jsonl' | head
```

再检查 Codex：

```bash
ls ~/.codex/sessions 2>/dev/null || true
find ~/.codex/sessions -name '*.jsonl' | head
```

如果至少一边能看到 JSONL 文件，`cchat` 就能开始工作。

---

### 4.4 macOS 弹窗依赖

如果你要使用弹窗模式，再检查：

```bash
which osascript
which pbcopy
```

正常情况下应看到：

```bash
/usr/bin/osascript
/usr/bin/pbcopy
```

如果只是终端模式，这两个即使有问题，也不一定影响 `cchat "关键词"` 这类命令。

## 5. `cchat` 的两种工作模式

## 5.1 弹窗模式

无参数执行：

```bash
cchat
```

或显式指定：

```bash
cchat --popup
```

弹窗模式适合：

- 不想记命令参数
- 想边看预览边选会话
- 想直接复制恢复命令
- 想一键在 Terminal 中恢复会话

### 弹窗模式流程

1. 打开输入框
2. 你可以：
   - 输入关键词并点击“搜索”
   - 点击“更多方式”
3. 在“更多方式”中可以选：
   - 最近聊天
   - 按日期
4. 系统会显示会话列表
5. 你选中一个会话后，会看到详情
6. 可以执行：
   - 复制恢复命令
   - 在终端打开

默认情况下，这些结果可能同时来自 Claude Code 和 Codex，列表中会显示来源标识。

---

## 5.2 终端模式

只要你给 `cchat` 传参数，它就会直接走终端模式。

例如：

```bash
cchat "飞书"
cchat --list
cchat --list --date 2026-04-15
cchat "claude code" --date 2026-04-15
cchat --source codex --list
```

终端模式适合：

- 你已经在命令行工作流中
- 想快速搜一把
- 想用 `--json` 输出给别的脚本继续处理
- 想用 `--source` 精确限定来源

---

## 7. 输出结果怎么看

## 8. 换一台电脑，如何从 0 开始安装

## 8.1 一个关键事实

虽然以后你主要只使用 `cchat` 这一个命令，但安装时仍然需要两个脚本文件：

- `cchat`
- `csearch`

原因是：

- `cchat` 是统一入口
- `csearch` 是它背后的搜索引擎

---

## 8.2 最推荐的安装方式：一键安装

在新电脑上，只要项目已经发布到 GitHub，直接执行：

```bash
curl -fsSL https://raw.githubusercontent.com/askfanxiaojun/cchat/main/install.sh | bash
```

安装器会自动完成这些事情：

- 下载 `cchat`
- 下载 `csearch`
- 安装到 `~/.local/bin`
- 自动补 PATH（如果还没配）
- 做基本安装提示

安装后，重新打开终端，或者执行：

```bash
source ~/.zshrc
```

如果你使用 `bash`：

```bash
source ~/.bashrc
```

---

## 8.3 新电脑安装前提

新电脑上至少要满足：

1. 已安装 macOS（如果你想用弹窗模式）
2. 已安装 Python 3
3. 已安装 Claude Code 或 Codex 中的任意一个，或两个都装
4. 这台机器上至少已经实际使用过一次对应客户端，生成过本地会话目录
5. 终端可用

先检查：

```bash
python3 --version
which claude || true
which codex || true
find ~/.claude/projects -name '*.jsonl' | head
find ~/.codex/sessions -name '*.jsonl' | head
```

如果两个目录都完全没有记录，先在这台机器上实际使用一次 Claude Code 或 Codex，再回来搜索。

---

## 8.4 手工安装方式（不用一键脚本时）

### 第一步：创建目录

```bash
mkdir -p ~/.local/bin
```

### 第二步：放入两个脚本

需要放入：

- `cchat`
- `csearch`

无论你是：

- 从旧电脑复制
- 从 GitHub 下载
- 或者手工粘贴源码

目标位置都应是：

```bash
~/.local/bin
```

### 第三步：加执行权限

```bash
chmod +x ~/.local/bin/cchat ~/.local/bin/csearch
```

### 第四步：把 `~/.local/bin` 加进 PATH

如果你用的是 zsh：

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

如果你用的是 bash：

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

### 第五步：验证命令是否生效

```bash
which cchat
which csearch
```

### 第六步：验证脚本语法

```bash
python3 -m py_compile ~/.local/bin/cchat ~/.local/bin/csearch
```

### 第七步：验证终端模式

```bash
cchat --list --max 3
```

### 第八步：验证弹窗模式

```bash
cchat
```

## 10. 常见问题排查

### 10.1 `command not found: cchat`

说明终端找不到命令。

检查：

```bash
ls -l ~/.local/bin/cchat ~/.local/bin/csearch
which cchat
```

修复方式通常是：

```bash
chmod +x ~/.local/bin/cchat ~/.local/bin/csearch
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

---

### 10.2 `cchat --list` 没有任何结果

检查本地是否确实存在记录：

```bash
find ~/.claude/projects -name '*.jsonl' | head -20
find ~/.codex/sessions -name '*.jsonl' | head -20
```

如果都没有输出：

- 这台机器还没有实际使用过 Claude Code / Codex
- 或本地记录已经被清理

---

### 10.3 想只看 Claude Code 或只看 Codex

使用：

```bash
cchat --source claude --list
cchat --source codex --list
```

或：

```bash
cchat "关键词" --source claude
cchat "关键词" --source codex
```

---

### 10.4 `cchat` 弹窗打不开

先检查：

```bash
which osascript
```

如果当前环境不是正常的 macOS 图形环境，弹窗模式可能失败。

但终端模式通常仍可继续使用：

```bash
cchat "关键词"
```

---

### 10.5 点击“在终端打开”后恢复失败

因为现在支持两类会话，所以要检查对应客户端是否存在：

```bash
which claude || true
which codex || true
```

规则是：

- Claude Code 会话 -> 需要 `claude`
- Codex 会话 -> 需要 `codex`

---

### 10.6 日期格式错误

日期必须写成：

```text
YYYY-MM-DD
```

例如：

- `2026-04-15`
- `2025-12-01`

不能写成：

- `2026/04/15`
- `04-15-2026`
- `15-04-2026`

---

### 10.7 我还能不能直接用 `csearch`

可以。

但建议的心智模型还是：

- 平时只记 `cchat`
- `csearch` 是内部引擎或高级调试入口

```python
def bundled_binary(name: str) -> Path:
    return Path(__file__).resolve().with_name(name)


def find_binary(name: str, fallback: Path) -> str:
    local = bundled_binary(name)
    if local.exists():
        return str(local)
    return shutil.which(name) or str(fallback)


def run_applescript(script: str, *args: str) -> str | None:
    result = subprocess.run(
        ["osascript", "-", *args],
        input=script,
        text=True,
        capture_output=True,
    )
    if result.returncode != 0:
        stderr = (result.stderr or "").strip()
        if "(-128)" in stderr:
            return None
        raise RuntimeError(stderr or "osascript failed")
    output = result.stdout.rstrip("\n")
    if output == "__CANCEL__":
        return None
    return output


def run_cli_mode(csearch: str, argv: list[str]) -> int:
    env = os.environ.copy()
    env["CSEARCH_PROG"] = "cchat"
    result = subprocess.run([csearch, *argv], env=env)
    return result.returncode


def truncate(text: str, limit: int) -> str:
    compact = " ".join(str(text).split())
    if len(compact) <= limit:
        return compact
    return compact[: max(0, limit - 1)] + "…"


def run_csearch(
    csearch: str, query: str, list_mode: bool, date_text: str = ""
) -> list[dict[str, object]]:
    cmd = [
        csearch,
        "--json",
        "--no-color",
        "--max",
        str(MAX_RESULTS),
        "--snippets",
        str(SNIPPETS),
    ]
    if date_text:
        cmd.extend(["--date", date_text])
    if list_mode:
        cmd.append("--list")
        if query:
            cmd.append(query)
    else:
        cmd.append(query)

    result = subprocess.run(cmd, text=True, capture_output=True)
    stdout = result.stdout.strip()
    if stdout:
        return json.loads(stdout)
    if result.returncode == 0:
        return []
    raise RuntimeError((result.stderr or "csearch failed").strip())


def build_choice(index: int, item: dict[str, object]) -> str:
    updated = item.get("updated_at_display", "")
    title = truncate(item.get("title", ""), TITLE_LIMIT)
    project = item.get("project", "")
    source = item.get("source_display") or item.get("source") or "unknown"
    return f"{index:02d}. [{source}] {updated} | {title} | {project}"


def build_detail(item: dict[str, object], query: str, list_mode: bool) -> str:
    lines = [
        f"标题: {item.get('title', '')}",
        f"来源: {item.get('source_display', item.get('source', ''))}",
        f"时间: {item.get('updated_at_display', '')}",
        f"项目: {item.get('project', '')}",
        (
            f"目录: {item.get('cwd', '')}"
            if item.get("cwd")
            else None
        ),
        f"Session: {item.get('session_id', '')}",
        f"恢复命令: {item.get('resume_command', '')}",
        "",
    ]
    lines = [line for line in lines if line is not None]

    snippets = item.get("snippets") or []
    if snippets:
        lines.append("预览:")
        for snippet in snippets:
            label = snippet.get("label", "")
            text = truncate(snippet.get("text", ""), PREVIEW_LIMIT)
            lines.append(f"[{label}] {text}")
    elif item.get("last_prompt"):
        lines.append("最近提示:")
        lines.append(truncate(item.get("last_prompt", ""), PREVIEW_LIMIT))
    elif list_mode:
        lines.append("这个会话目前没有可展示的预览内容。")
    else:
        lines.append(f'没有找到和 "{query}" 对应的预览片段。')

    return "\n".join(lines)


def choose_date(default_text: str) -> str | None:
    result = run_applescript(DATE_DIALOG, default_text)
    if result is None:
        return None
    if result == "__TODAY__" or not result.strip():
        return datetime.now().astimezone().strftime("%Y-%m-%d")

    date_text = result.strip()
    try:
        datetime.strptime(date_text, "%Y-%m-%d")
    except ValueError:
        run_applescript(ALERT_DIALOG, "日期格式不对，请使用 YYYY-MM-DD。")
        return choose_date(default_text)
    return date_text


def popup_main(default_query: str) -> int:
    csearch = find_binary("csearch", Path.home() / ".local" / "bin" / "csearch")

    answer = run_applescript(INPUT_DIALOG, default_query)
    if answer is None:
        return 0

    parts = answer.splitlines()
    action = parts[0].strip() if parts else "搜索"
    query = parts[1].strip() if len(parts) > 1 else default_query
    if action == "更多方式":
        action = run_applescript(MODE_DIALOG) or "取消"
        if action == "取消":
            return 0
    date_text = ""
    if action == "按日期":
        date_text = choose_date(datetime.now().astimezone().strftime("%Y-%m-%d"))
        if date_text is None:
            return 0
    list_mode = action == "最近聊天" or not query
    if action == "按日期":
        list_mode = True

    try:
        items = run_csearch(csearch, query, list_mode=list_mode, date_text=date_text)
    except Exception as exc:  # noqa: BLE001
        run_applescript(ALERT_DIALOG, f"读取聊天记录失败：{exc}")
        return 1

    if not items:
        if date_text:
            message = f"没有找到 {date_text} 这一天的聊天记录。"
        else:
            message = "没有找到最近聊天记录。" if list_mode else f'没有找到包含“{query}”的聊天记录。'
        run_applescript(ALERT_DIALOG, message)
        return 0

    choices = [build_choice(idx, item) for idx, item in enumerate(items, start=1)]
    chosen = run_applescript(PICK_DIALOG, *choices)
    if chosen is None:
        return 0

    selected_index = choices.index(chosen)
    selected = items[selected_index]
    detail = build_detail(selected, query, list_mode)
    follow_up = run_applescript(DETAIL_DIALOG, detail)
    if follow_up is None or follow_up == "取消":
        return 0

    resume_cmd = str(selected.get("resume_command", "")).strip()
    if follow_up == "复制恢复命令":
        subprocess.run(["pbcopy"], input=resume_cmd, text=True, check=False)
        run_applescript(ALERT_DIALOG, "恢复命令已经复制到剪贴板。")
        return 0

    if follow_up == "在终端打开":
        run_applescript(OPEN_TERMINAL, resume_cmd)
    return 0


def main() -> int:
    csearch = find_binary("csearch", Path.home() / ".local" / "bin" / "csearch")
    argv = sys.argv[1:]

    if not argv:

    if argv[0] in HELP_FLAGS:

        return 0

    if argv[0] in GUI_FLAGS:

    return run_cli_mode(csearch, argv)


if __name__ == "__main__":
    sys.exit(main())
```

```python
@dataclass

    text: str

def use_color(args: argparse.Namespace) -> bool:
    return sys.stdout.isatty() and not args.no_color


def colorize(text: str, code: str, enabled: bool) -> str:
    if not enabled:
        return text
    return f"\033[{code}m{text}\033[0m"


def normalize_whitespace(text: object) -> str:
    return " ".join(str(text).split())


def parse_timestamp(raw: object) -> Optional[datetime]:
    if not isinstance(raw, str) or not raw:
        return None
    try:
        return datetime.fromisoformat(raw.replace("Z", "+00:00")).astimezone()
    except ValueError:
        return None


def collect_claude_text_blocks(content: object) -> Iterable[str]:
    if isinstance(content, str):
        yield content
        return

    if isinstance(content, list):
        for block in content:
            if not isinstance(block, dict):
                continue
            if block.get("type") == "text":
                text = block.get("text")
                if isinstance(text, str) and text.strip():
                    yield text
        return

    if isinstance(content, dict) and content.get("type") == "text":
        text = content.get("text")
        if isinstance(text, str) and text.strip():
            yield text


def collect_codex_text_blocks(content: object) -> Iterable[str]:
    if not isinstance(content, list):
        return
    for block in content:
        if not isinstance(block, dict):
            continue
        if block.get("type") not in {"input_text", "output_text"}:
            continue
        text = block.get("text")
        if isinstance(text, str) and text.strip():
            yield text


def truncate_plain(text: str, limit: int) -> str:
    text = normalize_whitespace(text)
    if len(text) <= limit:
        return text
    return text[: max(0, limit - 1)] + "…"


def fallback_title(session_id: str, last_prompt: str, messages: list[MessageEntry]) -> str:
    if last_prompt:
        return truncate_plain(last_prompt, 56)
    if messages:
        return truncate_plain(messages[0].text, 56)
    return f"(untitled) {session_id[:8]}"


def format_time(value: datetime) -> str:
    return value.astimezone().strftime("%Y-%m-%d %H:%M")


def build_resume_command(source: str, session_id: str) -> str:
    if source == "codex":
        return f"codex resume {session_id}"
    return f"claude --resume {session_id}"


def local_date(value: datetime) -> date:
    return value.astimezone().date()


def source_label(source: str) -> str:
    if source == "codex":
        return "Codex"
    return "Claude"


def project_name_from_cwd(cwd: str) -> str:
    if not cwd:
        return "(unknown)"
    path = Path(cwd).expanduser()
    name = path.name.strip()
    return name or str(path)


def load_claude_session(path: Path) -> Optional[SessionEntry]:
    try:
        stat = path.stat()
    except OSError:
        return None

    project = path.parent.name
    session_id = path.stem
    fallback_time = datetime.fromtimestamp(stat.st_mtime, tz=timezone.utc).astimezone()
    updated_at = fallback_time
    title = ""
    last_prompt = ""
    messages: list[MessageEntry] = []

    try:
        with path.open("r", encoding="utf-8", errors="replace") as handle:
            for raw_line in handle:
                line = raw_line.strip()
                if not line:
                    continue
                try:
                    item = json.loads(line)
                except json.JSONDecodeError:
                    continue

                record_type = item.get("type")
                record_time = parse_timestamp(item.get("timestamp"))
                if record_time and record_time > updated_at:
                    updated_at = record_time

                if record_type == "custom-title":
                    value = normalize_whitespace(item.get("customTitle", ""))
                    if value:
                        title = value
                    continue

                if record_type == "last-prompt":
                    value = normalize_whitespace(item.get("lastPrompt", ""))
                    if value:
                        last_prompt = value
                    continue

                if record_type not in {"user", "assistant"}:
                    continue

                if item.get("isMeta"):
                    continue

                content = (item.get("message") or {}).get("content")
                text = normalize_whitespace(" ".join(collect_claude_text_blocks(content)))
                if not text:
                    continue
                messages.append(MessageEntry(role=record_type, text=text, timestamp=record_time))
    except OSError:
        return None

    return SessionEntry(
        source="claude",
        session_id=session_id,
        project=project,
        path=path,
        title=title or fallback_title(session_id, last_prompt, messages),
        updated_at=updated_at,
        resume_command=build_resume_command("claude", session_id),
        last_prompt=last_prompt,
        messages=messages,
    )


def load_codex_session(path: Path) -> Optional[SessionEntry]:
    try:
        stat = path.stat()
    except OSError:
        return None

    session_id = path.stem
    fallback_time = datetime.fromtimestamp(stat.st_mtime, tz=timezone.utc).astimezone()
    updated_at = fallback_time
    title = ""
    last_prompt = ""
    cwd = ""
    messages: list[MessageEntry] = []

    try:
        with path.open("r", encoding="utf-8", errors="replace") as handle:
            for raw_line in handle:
                line = raw_line.strip()
                if not line:
                    continue
                try:
                    item = json.loads(line)
                except json.JSONDecodeError:
                    continue

                record_time = parse_timestamp(item.get("timestamp"))
                if record_time and record_time > updated_at:
                    updated_at = record_time

                record_type = item.get("type")
                payload = item.get("payload") or {}

                if record_type == "session_meta":
                    payload_id = payload.get("id")
                    if isinstance(payload_id, str) and payload_id:
                        session_id = payload_id
                    payload_time = parse_timestamp(payload.get("timestamp"))
                    if payload_time and payload_time > updated_at:
                        updated_at = payload_time
                    value = payload.get("cwd")
                    if isinstance(value, str) and value.strip():
                        cwd = value.strip()
                    continue

                if record_type != "response_item":
                    continue
                if payload.get("type") != "message":
                    continue

                role = payload.get("role")
                if role not in {"user", "assistant"}:
                    continue

                text = normalize_whitespace(
                    " ".join(collect_codex_text_blocks(payload.get("content")))
                )
                if not text:
                    continue

                if role == "user" and not last_prompt:
                    last_prompt = text
                if role == "user" and not title:
                    title = truncate_plain(text, 56)
                messages.append(MessageEntry(role=role, text=text, timestamp=record_time))
    except OSError:
        return None

    project = project_name_from_cwd(cwd)
    return SessionEntry(
        source="codex",
        session_id=session_id,
        project=project,
        path=path,
        title=title or fallback_title(session_id, last_prompt, messages),
        updated_at=updated_at,
        resume_command=build_resume_command("codex", session_id),
        cwd=cwd,
        last_prompt=last_prompt,
        messages=messages,
    )


def iter_claude_sessions(root: Path, project_filter: Optional[str]) -> Iterable[SessionEntry]:
    if not root.exists():
        return

    project_match = project_filter.casefold() if project_filter else None
    for path in sorted(root.rglob("*.jsonl")):
        if not path.is_file():
            continue
        session = load_claude_session(path)
        if not session:
            continue
        if project_match and project_match not in session.project.casefold():
            continue
        yield session


def iter_codex_sessions(root: Path, project_filter: Optional[str]) -> Iterable[SessionEntry]:
    if not root.exists():
        return

    project_match = project_filter.casefold() if project_filter else None
    for path in sorted(root.rglob("*.jsonl")):
        if not path.is_file():
            continue
        session = load_codex_session(path)
        if not session:
            continue
        if project_match:
            haystacks = [session.project.casefold(), session.cwd.casefold()]
            if not any(project_match in item for item in haystacks if item):
                continue
        yield session


def iter_sessions(
    source: str, claude_root: Path, codex_root: Path, project_filter: Optional[str]
) -> Iterable[SessionEntry]:
    if source in {"all", "claude"}:
        yield from iter_claude_sessions(claude_root, project_filter)
    if source in {"all", "codex"}:
        yield from iter_codex_sessions(codex_root, project_filter)


def compile_pattern(query: str) -> re.Pattern[str]:
    return re.compile(re.escape(query), re.IGNORECASE)


def build_snippet(text: str, pattern: re.Pattern[str], colors: bool) -> str:
    text = normalize_whitespace(text)
    match = pattern.search(text)
    if not match:
        return truncate_plain(text, SNIPPET_WIDTH * 2)

    start = max(0, match.start() - SNIPPET_WIDTH)
    end = min(len(text), match.end() + SNIPPET_WIDTH)
    prefix = "…" if start > 0 else ""
    suffix = "…" if end < len(text) else ""
    snippet = prefix + text[start:end] + suffix

    def repl(found: re.Match[str]) -> str:
        return colorize(found.group(0), "1;31", colors)

    return pattern.sub(repl, snippet)


def session_to_dict(session: SessionEntry) -> dict[str, object]:
    return {
        "source": session.source,
        "source_display": source_label(session.source),
        "session_id": session.session_id,
        "project": session.project,
        "cwd": session.cwd,
        "path": str(session.path),
        "title": session.title,
        "updated_at": session.updated_at.astimezone().isoformat(),
        "updated_at_display": format_time(session.updated_at),
        "last_prompt": session.last_prompt,
        "resume_command": session.resume_command,
    }


def filter_by_date(sessions: list[SessionEntry], date_text: Optional[str]) -> list[SessionEntry]:
    if not date_text:
        return list(sessions)
    target = datetime.strptime(date_text, "%Y-%m-%d").date()
    return [session for session in sessions if local_date(session.updated_at) == target]


def print_session_header(
    session: SessionEntry, index: int, total: int, colors: bool, show_path: bool
) -> None:
    print(colorize(f"[{index}/{total}] {session.title}", "1;36", colors))
    print(f"  Source:  {source_label(session.source)}")
    print(f"  Time:    {format_time(session.updated_at)}")
    print(f"  Project: {session.project}")
    if session.cwd:
        print(f"  Cwd:     {session.cwd}")
    print(f"  Session: {session.session_id}")
    print(f"  Resume:  {session.resume_command}")
    if show_path:
        print(f"  Path:    {session.path}")


def list_session_rows(
    sessions: list[SessionEntry], query: Optional[str]
) -> list[SessionEntry]:
    if not query:
        return list(sessions)

    pattern = compile_pattern(query)
    return [
        session
        for session in sessions
        if pattern.search(session.title)
        or pattern.search(session.last_prompt)
        or pattern.search(session.session_id)
        or pattern.search(session.project)
        or pattern.search(session.source)
        or (session.cwd and pattern.search(session.cwd))
    ]


def search_session_rows(
    sessions: list[SessionEntry], query: str, snippets_limit: int, colors: bool
) -> list[tuple[SessionEntry, list[tuple[str, str]]]]:
    pattern = compile_pattern(query)
    matches: list[tuple[SessionEntry, list[tuple[str, str]]]] = []

    for session in sessions:
        snippets: list[tuple[str, str]] = []
        seen_keys: set[tuple[str, str]] = set()

        header_sources = [
            ("title", session.title),
            ("prompt", session.last_prompt),
            ("cwd", session.cwd),
        ]
        for label, text in header_sources:
            if text and pattern.search(text):
                snippet = build_snippet(text, pattern, colors)
                key = (label, ANSI_RE.sub("", snippet))
                if key not in seen_keys:
                    snippets.append((label, snippet))
                    seen_keys.add(key)
                if len(snippets) >= snippets_limit:
                    break

        if len(snippets) < snippets_limit:
            for message in session.messages:
                if not pattern.search(message.text):
                    continue
                label = "user" if message.role == "user" else "assistant"
                snippet = build_snippet(message.text, pattern, colors)
                key = (label, ANSI_RE.sub("", snippet))
                if key in seen_keys:
                    continue
                snippets.append((label, snippet))
                seen_keys.add(key)
                if len(snippets) >= snippets_limit:
                    break

        if snippets:
            matches.append((session, snippets))

    matches.sort(key=lambda item: item[0].updated_at, reverse=True)
    return matches


def list_sessions(sessions: list[SessionEntry], args: argparse.Namespace, colors: bool) -> int:
    filtered = list_session_rows(sessions, args.query)
    filtered.sort(key=lambda item: item.updated_at, reverse=True)
    limited = filtered[: args.max]
    if not limited:
        if args.json:
            print("[]")
        else:
            print("No sessions found.")
        return 1

    if args.json:
        print(
            json.dumps(
                [session_to_dict(session) for session in limited],
                ensure_ascii=False,
                indent=2,
            )
        )
        return 0

    for idx, session in enumerate(limited, start=1):
        if idx > 1:
            print()
        print_session_header(session, idx, len(limited), colors, args.path)
        if session.last_prompt:
            print(f"  Prompt:  {truncate_plain(session.last_prompt, 100)}")
    return 0


def search_sessions(sessions: list[SessionEntry], args: argparse.Namespace, colors: bool) -> int:
    matches = search_session_rows(sessions, args.query, args.snippets, colors)
    limited = matches[: args.max]
    if not limited:
        if args.json:
            print("[]")
        else:
            print(f'No matches for "{args.query}".')
        return 1

    if args.json:
        payload = []
        for session, snippets in limited:
            item = session_to_dict(session)
            item["snippets"] = [
                {"label": label, "text": ANSI_RE.sub("", snippet)}
                for label, snippet in snippets
            ]
            payload.append(item)
        print(json.dumps(payload, ensure_ascii=False, indent=2))
        return 0

    for idx, (session, snippets) in enumerate(limited, start=1):
        if idx > 1:
            print()
        print_session_header(session, idx, len(limited), colors, args.path)
        for label, snippet in snippets:
            print(f"  {label:>9}: {snippet}")
    return 0


def main() -> int:
    args = parse_args()
    colors = use_color(args)
    claude_root = Path(args.claude_root or args.root).expanduser()
    codex_root = Path(args.codex_root or DEFAULT_CODEX_ROOT).expanduser()
    sessions = list(iter_sessions(args.source, claude_root, codex_root, args.project))
    sessions = filter_by_date(sessions, args.date)
    sessions.sort(key=lambda item: item.updated_at, reverse=True)

    if args.list:
        return list_sessions(sessions, args, colors)
    return search_sessions(sessions, args, colors)


if __name__ == "__main__":
    sys.exit(main())
```

## 14. 一句话总结

- 现在你主要只需要记住一个命令：`cchat`
- `cchat` 默认同时搜索 Claude Code 和 Codex
- 需要时可以用 `--source claude` 或 `--source codex` 精确过滤
- `csearch` 仍然存在，但主要作为内部搜索引擎
- 换电脑时，最简单的方式是一键安装
