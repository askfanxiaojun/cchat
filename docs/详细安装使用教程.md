# cchat 详细安装与使用教程

> 这份文档对应当前最新版脚本：以后你主要只需要记住一个命令：`cchat`。
>
> - `cchat` 是你对外使用的主入口
> - `csearch` 仍然存在，但现在更适合作为内部搜索引擎或调试工具，不再是日常必须记忆的命令

---

## 1. 先说结论：现在你应该怎么用

如果你只想记住最少的内容，只记下面这几条就够了：

```bash
cchat
cchat "关键词"
cchat --list
cchat --list --date 2026-04-15
cchat --popup
cchat --popup "关键词"
```

它们分别表示：

- `cchat`：打开 macOS 弹窗界面
- `cchat "关键词"`：直接在终端里搜索聊天记录
- `cchat --list`：在终端里列出最近聊天记录
- `cchat --list --date 2026-04-15`：列出某一天的聊天记录
- `cchat --popup`：强制打开弹窗界面
- `cchat --popup "关键词"`：打开弹窗，并把关键词预先填进去

如果你平时只是为了“找以前聊过的内容”，基本已经够用了。

### GitHub 一键安装

如果这个项目已经发布到 GitHub，你在新电脑上可以直接执行：

```bash
curl -fsSL https://raw.githubusercontent.com/askfanxiaojun/cchat/main/install.sh | bash
```

安装完成后，重新打开终端，或者执行：

```bash
source ~/.zshrc
```

## 2. `cchat` 和 `csearch` 现在分别是什么

### 2.1 `cchat` 是什么

`cchat` 是你现在唯一需要记住的外部命令。

它同时支持两种工作方式：

1. 图形弹窗模式
2. 终端命令行模式

也就是说，现在它既可以像一个小工具一样弹窗，也可以像命令行搜索器一样直接在终端里输出结果。

你可以把它理解成：

- 一个统一入口
- 一个对 Claude Code 本地聊天记录的搜索与恢复助手

---

### 2.2 `csearch` 是什么

`csearch` 现在更像是 `cchat` 背后的内部搜索引擎。

它负责这些底层工作：

- 读取 Claude Code 保存在本地的 JSONL 会话文件
- 解析标题、时间、消息内容、最后一条 prompt
- 按关键词搜索
- 按日期过滤
- 列出最近会话
- 输出结构化 JSON 数据

你可以继续直接使用它，但从现在开始，不建议再把它当作日常主命令来记忆。

更好的理解方式是：

- `cchat`：面向日常使用
- `csearch`：面向内部实现、调试、复用

---

### 2.3 为什么现在改成只记一个命令

之前分成两个名字：

- `csearch`
- `cchat`

从技术上没问题，但从使用体验看，确实增加了记忆成本。

所以现在改成：

- 对外统一记 `cchat`
- 对内依然保留 `csearch` 做底层能力

这样做的好处：

- 你平时更省脑力
- 文档更简单
- 换电脑时也更清楚该怎么装
- 以后如果继续扩展，也只需要围绕 `cchat` 这个入口展开

---

## 3. 这些工具到底在读什么数据

这套工具不是联网去查历史消息，而是直接读取 Claude Code 保存在本机的会话文件。

默认目录是：

```bash
~/.claude/projects
```

这些目录下通常会有很多 `.jsonl` 文件。每个 `.jsonl` 文件对应一个 Claude Code 会话，里面一行一条 JSON 记录，可能包含：

- 用户消息
- 助手消息
- 工具调用
- 会话标题
- 最后一次 prompt
- 时间戳

`csearch`/`cchat` 会从中提取对搜索最有用的部分，例如：

- `custom-title`
- `last-prompt`
- `user`
- `assistant`

所以，**前提不是“你联网能访问 Claude”，而是“你这台机器上确实存有 Claude Code 的本地会话文件”**。

## 4. 使用前必须先确认什么

正式使用之前，建议先检查 4 件事。

### 4.1 检查 Python 3

```bash
python3 --version
```

如果能看到版本号，例如：

```bash
Python 3.9.6
```

说明 Python 3 可以用。

---

### 4.2 检查 Claude Code 是否安装

```bash
which claude
```

如果能看到类似输出：

```bash
/Users/你的用户名/.local/bin/claude
```

说明 `claude` 命令可用。

---

### 4.3 检查 Claude Code 本地会话目录是否存在

```bash
ls ~/.claude/projects
```

如果目录存在，再继续检查是否真的有 JSONL 文件：

```bash
find ~/.claude/projects -name '*.jsonl' | head
```

如果这条命令能输出内容，说明你的本地历史记录存在，`cchat`/`csearch` 就有数据可读。

---

### 4.4 检查 macOS 图形组件是否可用

因为 `cchat` 的弹窗模式依赖 macOS 自带组件，所以还建议检查：

```bash
which osascript
which pbcopy
```

正常情况下应当能看到：

```bash
/usr/bin/osascript
/usr/bin/pbcopy
```

如果你只是想用终端模式，那么就算图形组件有问题，`cchat "关键词"` 这类用法仍然可能继续工作。

## 5. `cchat` 的两种工作模式

## 5.1 模式 A：弹窗模式

无参数执行：

```bash
cchat
```

或者显式指定弹窗模式：

```bash
cchat --popup
```

这时会走 macOS 弹窗流程。

弹窗流程是这样的：

1. 先出现一个输入框
2. 你可以：
   - 直接输入关键词并点击“搜索”
   - 点击“更多方式”
3. 如果进入“更多方式”，可以继续选：
   - 最近聊天
   - 按日期
4. 选中会话后，会出现详情窗口
5. 你可以：
   - 复制恢复命令
   - 在 Terminal 中打开该会话

### 弹窗模式适合谁

适合这些场景：

- 不想记参数
- 想用鼠标点选
- 想先看预览再决定恢复哪条会话
- 临时快速查一下之前聊过什么

---

## 5.2 模式 B：终端模式

只要你给 `cchat` 传参数，它就会进入终端模式，而不是弹窗。

例如：

```bash
cchat "飞书"
cchat --list
cchat --list --date 2026-04-15
cchat "claude code" --date 2026-04-15
```

这时它会直接把结果打印到终端。

### 终端模式适合谁

适合这些场景：

- 你已经在终端工作流里
- 你想快速复制输出
- 你想把输出继续管道给别的命令
- 你想用 `--json` 做二次处理

---

## 6. `cchat` 日常最常用命令详解

### 6.1 打开弹窗

```bash
cchat
```

用途：最省心的默认入口。

适合：你不确定该搜什么，或者想边看边选。

---

### 6.2 弹窗并预填关键词

```bash
cchat --popup "claude code"
```

用途：打开弹窗，但把输入框先填上默认关键词。

适合：你已经大概知道要搜什么，但仍然想通过弹窗慢慢挑。

---

### 6.3 直接搜索关键词

```bash
cchat "关键词"
```

例如：

```bash
cchat "飞书"
cchat "nano-banana"
cchat "claude code"
```

用途：在终端直接搜索所有会话内容。

输出里通常会看到：

- 标题
- 时间
- 项目
- Session ID
- 恢复命令
- 命中的预览片段

---

### 6.4 列出最近会话

```bash
cchat --list
```

用途：不按关键词，只看最近有哪些会话。

适合：你只记得“最近聊过”，但忘了具体关键词。

---

### 6.5 查看某天的会话

```bash
cchat --list --date 2026-04-15
```

用途：只看某一天更新过的会话。

注意日期格式必须是：

```text
YYYY-MM-DD
```

例如：

- `2026-04-15`
- `2025-12-01`

不能写成：

- `2026/04/15`
- `04-15-2026`
- `15-04-2026`

---

### 6.6 搜索某天的关键词

```bash
cchat "claude code" --date 2026-04-15
```

用途：先限定日期，再按关键词搜索。

适合：你知道大概是哪天，但那天会话很多，想进一步缩小范围。

---

### 6.7 限制结果数量

```bash
cchat "关键词" --max 5
```

用途：最多显示 5 条会话。

适合：关键词特别常见时，避免结果太长。

---

### 6.8 限制每条会话显示几段预览

```bash
cchat "关键词" --snippets 2
```

用途：每个命中的会话只显示最多 2 条预览片段。

---

### 6.9 只看某个项目目录名匹配的会话

```bash
cchat "关键词" --project lark
```

用途：只在项目目录名包含 `lark` 的会话中搜索。

---

### 6.10 查看原始 JSON 结构结果

```bash
cchat "关键词" --json
```

用途：把结果输出为 JSON，而不是普通终端文本。

适合：

- 写脚本继续处理
- 导出数据
- 做更复杂的自动化

---

### 6.11 查看帮助

```bash
cchat --help
```

用途：快速提醒自己主用法。

---

## 7. 终端输出怎么看

在终端模式下，典型输出大概像这样：

```text
[1/1] 重试
  Time:    2026-04-15 14:45
  Project: -Users-fanxiaojun
  Session: 404948d1-ecdd-4af2-b892-68541d2a6bc8
  Resume:  claude --resume 404948d1-ecdd-4af2-b892-68541d2a6bc8
       user: 如何快速的搜索claude code的session？
```

这里最重要的是两部分：

- `Session:`：真正的会话 ID
- `Resume:`：已经帮你拼好的恢复命令

你只要复制这一行就能恢复：

```bash
claude --resume 404948d1-ecdd-4af2-b892-68541d2a6bc8
```

## 8. 换一台电脑，如何从 0 开始安装

这一节是最重要的。假设你现在拿到一台全新的 macOS 电脑，要把这套工具重新装起来。

## 8.1 你要知道一个关键点

虽然以后你主要只使用 `cchat` 这一个命令，**但安装时仍然需要两个脚本文件**：

- `cchat`
- `csearch`

原因是：

- `cchat` 是统一入口
- `csearch` 是它背后的搜索引擎

所以“使用上一个命令”，不等于“磁盘上只有一个文件”。

---

## 8.2 新电脑安装前提

在新电脑上，至少要满足：

1. 已安装 macOS
2. 已安装 Python 3
3. 已安装 Claude Code
4. 这台电脑上已经实际运行过 Claude Code，生成过本地会话目录
5. 终端可用

先检查：

```bash
python3 --version
which claude
ls ~/.claude/projects
```

如果 `~/.claude/projects` 根本不存在，请先在新电脑上用一次 Claude Code，再回来继续安装。

---

## 8.3 安装方式 A：直接从旧电脑复制脚本

这是最推荐的方法。

### 第一步：在旧电脑上确认文件位置

```bash
ls -l ~/.local/bin/cchat
ls -l ~/.local/bin/csearch
```

### 第二步：把文件复制到新电脑

可以用这些方式：

- AirDrop
- iCloud Drive
- 微信文件传输助手
- Git 仓库
- `scp`
- U 盘

假设你已经把两个脚本拷到了新电脑的 `~/Downloads`。

### 第三步：在新电脑创建目标目录

```bash
mkdir -p ~/.local/bin
```

### 第四步：移动文件到目标目录

```bash
mv ~/Downloads/cchat ~/.local/bin/cchat
mv ~/Downloads/csearch ~/.local/bin/csearch
```

### 第五步：加执行权限

```bash
chmod +x ~/.local/bin/cchat ~/.local/bin/csearch
```

---

## 8.4 安装方式 B：在新电脑手工创建脚本

如果你拿不到旧电脑文件，也可以完全手工恢复。

### 第一步：创建目录

```bash
mkdir -p ~/.local/bin
```

### 第二步：创建 `csearch`

先执行：

```bash
cat > ~/.local/bin/csearch <<'PY'
```

然后把本文档附录里的 `csearch` 源码完整粘贴进去，最后单独输入：

```bash
PY
```

### 第三步：创建 `cchat`

先执行：

```bash
cat > ~/.local/bin/cchat <<'PY'
```

然后把本文档附录里的 `cchat` 源码完整粘贴进去，最后单独输入：

```bash
PY
```

### 第四步：加执行权限

```bash
chmod +x ~/.local/bin/cchat ~/.local/bin/csearch
```

## 8.5 把 `~/.local/bin` 加入 PATH

很多新电脑安装完脚本后，最大的问题不是脚本写错了，而是终端根本找不到命令。

先检查当前 shell：

```bash
echo $SHELL
```

### 如果你用的是 zsh

macOS 默认通常是 zsh。

把下面这行加到 `~/.zshrc`：

```bash
export PATH="$HOME/.local/bin:$PATH"
```

直接执行：

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

### 如果你用的是 bash

可以写进 `~/.bashrc` 或 `~/.bash_profile`：

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

### 检查 PATH 是否生效

```bash
which cchat
which csearch
```

理想输出应类似：

```bash
/Users/你的用户名/.local/bin/cchat
/Users/你的用户名/.local/bin/csearch
```

虽然 `csearch` 也在 PATH 中，但以后你可以把它当成内部命令，不必专门记它。

## 10. 常见问题排查

### 10.1 终端提示 `command not found: cchat`

说明 PATH 没配好，或者文件不在正确位置。

先检查：

```bash
ls -l ~/.local/bin/cchat ~/.local/bin/csearch
which cchat
```

修复通常是：

```bash
chmod +x ~/.local/bin/cchat ~/.local/bin/csearch
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

---

### 10.2 `cchat --list` 没有结果

说明本地没有搜到可读的 Claude 会话。

检查：

```bash
find ~/.claude/projects -name '*.jsonl' | head -20
```

如果没有输出，通常是：

- 新电脑还没有实际使用过 Claude Code
- 会话目录不在默认位置
- 本地记录被清理过

这时可以：

- 先用一次 Claude Code
- 或者改用 `--root` 指定真实目录

例如：

```bash
cchat --list --root /你的/实际/路径
```

---

### 10.3 `cchat` 弹窗打不开

先检查：

```bash
which osascript
```

如果不是 macOS 或当前环境不支持图形窗口，弹窗模式可能失败。

但这不一定影响终端模式。你仍然可以先使用：

```bash
cchat "关键词"
```

---

### 10.4 `cchat` 详情里点“在终端打开”没有成功恢复

这通常说明 `claude` 命令本身不在 PATH 中。

检查：

```bash
which claude
```

如果找不到 `claude`，先修复 Claude Code 的安装或 PATH。

---

### 10.5 日期格式不对

日期必须写成：

```text
YYYY-MM-DD
```

例如：

- `2026-04-15`
- `2025-12-01`

不是：

- `2026/04/15`
- `04-15-2026`
- `15-04-2026`

---

### 10.6 我还能不能直接用 `csearch`

可以。

它没有被删掉，仍然可用。

但从现在开始，更推荐的心智模型是：

- 平时只记 `cchat`
- `csearch` 只是内部引擎或高级调试入口

如果你只是正常找聊天记录，不需要强迫自己记两个命令。

```python
  cchat "keyword"           Search sessions in the terminal
  cchat --list              List recent sessions in the terminal
  cchat --date YYYY-MM-DD --list
                            List sessions from a specific date
  cchat --popup             Force popup mode
  cchat --popup "keyword"   Open popup mode with a prefilled keyword

Notes:
  - cchat is the main command you need to remember.
  - The terminal mode forwards options to the internal search engine.
  - For full structured output, you can still use: cchat --json ...
"""

on run argv

        return "__CANCEL__"

end run
'''

DATE_DIALOG = r'''
on run argv
    set defaultDate to ""
    if (count of argv) > 0 then set defaultDate to item 1 of argv

def find_binary(name: str, fallback: Path) -> str:
    return shutil.which(name) or str(fallback)


def run_applescript(script: str, *args: str) -> str | None:
    result = subprocess.run(
        ["osascript", "-", *args],
        input=script,
        text=True,
        capture_output=True,
    )
    if result.returncode != 0:
        stderr = (result.stderr or "").strip()
        if "(-128)" in stderr:
            return None
        raise RuntimeError(stderr or "osascript failed")
    output = result.stdout.rstrip("\n")
    if output == "__CANCEL__":
        return None
    return output


def run_cli_mode(csearch: str, argv: list[str]) -> int:
    env = os.environ.copy()
    env["CSEARCH_PROG"] = "cchat"
    result = subprocess.run([csearch, *argv], env=env)
    return result.returncode


def truncate(text: str, limit: int) -> str:
    compact = " ".join(str(text).split())
    if len(compact) <= limit:
        return compact
    return compact[: max(0, limit - 1)] + "…"


def run_csearch(
    csearch: str, query: str, list_mode: bool, date_text: str = ""
) -> list[dict[str, object]]:
    cmd = [
        csearch,
        "--json",
        "--no-color",
        "--max",
        str(MAX_RESULTS),
        "--snippets",
        str(SNIPPETS),
    ]
    if date_text:
        cmd.extend(["--date", date_text])
    if list_mode:
        cmd.append("--list")
        if query:
            cmd.append(query)
    else:
        cmd.append(query)

    result = subprocess.run(cmd, text=True, capture_output=True)
    stdout = result.stdout.strip()
    if stdout:
        return json.loads(stdout)
    if result.returncode == 0:
        return []
    raise RuntimeError((result.stderr or "csearch failed").strip())


def build_choice(index: int, item: dict[str, object]) -> str:
    updated = item.get("updated_at_display", "")
    title = truncate(item.get("title", ""), TITLE_LIMIT)
    project = item.get("project", "")
    return f"{index:02d}. {updated} | {title} | {project}"


def build_detail(item: dict[str, object], query: str, list_mode: bool) -> str:
    lines = [
        f"标题: {item.get('title', '')}",
        f"时间: {item.get('updated_at_display', '')}",
        f"项目: {item.get('project', '')}",
        f"Session: {item.get('session_id', '')}",
        f"恢复命令: claude --resume {item.get('session_id', '')}",
        "",
    ]

    snippets = item.get("snippets") or []
    if snippets:
        lines.append("预览:")
        for snippet in snippets:
            label = snippet.get("label", "")
            text = truncate(snippet.get("text", ""), PREVIEW_LIMIT)
            lines.append(f"[{label}] {text}")
    elif item.get("last_prompt"):
        lines.append("最近提示:")
        lines.append(truncate(item.get("last_prompt", ""), PREVIEW_LIMIT))
    elif list_mode:
        lines.append("这个会话目前没有可展示的预览内容。")
    else:
        lines.append(f'没有找到和 "{query}" 对应的预览片段。')

    return "\n".join(lines)


def choose_date(default_text: str) -> str | None:
    result = run_applescript(DATE_DIALOG, default_text)
    if result is None:
        return None
    if result == "__TODAY__" or not result.strip():
        return datetime.now().astimezone().strftime("%Y-%m-%d")

    date_text = result.strip()
    try:
        datetime.strptime(date_text, "%Y-%m-%d")
    except ValueError:
        run_applescript(ALERT_DIALOG, "日期格式不对，请使用 YYYY-MM-DD。")
        return choose_date(default_text)
    return date_text


def popup_main(default_query: str) -> int:
    csearch = find_binary("csearch", Path.home() / ".local" / "bin" / "csearch")
    claude = find_binary("claude", Path.home() / ".local" / "bin" / "claude")

    answer = run_applescript(INPUT_DIALOG, default_query)
    if answer is None:
        return 0

    parts = answer.splitlines()
    action = parts[0].strip() if parts else "搜索"
    query = parts[1].strip() if len(parts) > 1 else default_query
    if action == "更多方式":
        action = run_applescript(MODE_DIALOG) or "取消"
        if action == "取消":
            return 0
    date_text = ""
    if action == "按日期":
        date_text = choose_date(datetime.now().astimezone().strftime("%Y-%m-%d"))
        if date_text is None:
            return 0
    list_mode = action == "最近聊天" or not query
    if action == "按日期":
        list_mode = True

    try:
        items = run_csearch(csearch, query, list_mode=list_mode, date_text=date_text)
    except Exception as exc:  # noqa: BLE001
        run_applescript(ALERT_DIALOG, f"读取聊天记录失败：{exc}")
        return 1

    if not items:
        if date_text:
            message = f"没有找到 {date_text} 这一天的聊天记录。"
        else:
            message = "没有找到最近聊天记录。" if list_mode else f'没有找到包含“{query}”的聊天记录。'
        run_applescript(ALERT_DIALOG, message)
        return 0

    choices = [build_choice(idx, item) for idx, item in enumerate(items, start=1)]
    chosen = run_applescript(PICK_DIALOG, *choices)
    if chosen is None:
        return 0

    selected_index = choices.index(chosen)
    selected = items[selected_index]
    detail = build_detail(selected, query, list_mode)
    follow_up = run_applescript(DETAIL_DIALOG, detail)
    if follow_up is None or follow_up == "取消":
        return 0

    resume_cmd = f"{claude} --resume {selected.get('session_id', '')}"
    if follow_up == "复制恢复命令":
        subprocess.run(["pbcopy"], input=resume_cmd, text=True, check=False)
        run_applescript(ALERT_DIALOG, "恢复命令已经复制到剪贴板。")
        return 0

    if follow_up == "在终端打开":
        run_applescript(OPEN_TERMINAL, resume_cmd)
    return 0


def main() -> int:
    csearch = find_binary("csearch", Path.home() / ".local" / "bin" / "csearch")
    argv = sys.argv[1:]

    if not argv:
        return popup_main("")

    if argv[0] in HELP_FLAGS:
        print(HELP_TEXT)
        return 0

    if argv[0] in GUI_FLAGS:
        return popup_main(" ".join(argv[1:]).strip())

    return run_cli_mode(csearch, argv)


if __name__ == "__main__":
    sys.exit(main())
```

```python
#!/usr/bin/env python3
"""Search and preview Claude Code session transcripts."""

from __future__ import annotations

import argparse
import json
import os
import re
import sys
from dataclasses import dataclass, field
from datetime import date, datetime, timezone
from pathlib import Path
from typing import Iterable, Optional

DEFAULT_ROOT = Path.home() / ".claude" / "projects"
DEFAULT_MAX_RESULTS = 10
DEFAULT_SNIPPETS = 3
SNIPPET_WIDTH = 72
ANSI_RE = re.compile(r"\x1b\[[0-9;]*m")


@dataclass
class MessageEntry:
    role: str
    text: str
    timestamp: Optional[datetime] = None


@dataclass
class SessionEntry:
    session_id: str
    project: str
    path: Path
    title: str
    updated_at: datetime
    last_prompt: str = ""
    messages: list[MessageEntry] = field(default_factory=list)


def parse_args() -> argparse.Namespace:
    parser = argparse.ArgumentParser(
        prog=os.environ.get("CSEARCH_PROG", "csearch"),
        description="Search and preview Claude Code session JSONL files."
    )
    parser.add_argument("query", nargs="?", help="Search text.")
    parser.add_argument(
        "--root",
        default=str(DEFAULT_ROOT),
        help="Claude projects root directory. Default: ~/.claude/projects",
    )
    parser.add_argument(
        "--list",
        action="store_true",
        help="List sessions instead of searching message contents.",
    )
    parser.add_argument(
        "--project",
        help="Only include project directories whose name contains this text.",
    )
    parser.add_argument(
        "--max",
        type=int,
        default=DEFAULT_MAX_RESULTS,
        help=f"Maximum sessions to print. Default: {DEFAULT_MAX_RESULTS}",
    )
    parser.add_argument(
        "--snippets",
        type=int,
        default=DEFAULT_SNIPPETS,
        help=f"Maximum matching snippets to print per session. Default: {DEFAULT_SNIPPETS}",
    )
    parser.add_argument(
        "--no-color",
        action="store_true",
        help="Disable ANSI colors.",
    )
    parser.add_argument(
        "--path",
        action="store_true",
        help="Show the JSONL path for each result.",
    )
    parser.add_argument(
        "--date",
        help="Only include sessions updated on this local date, format: YYYY-MM-DD",
    )
    parser.add_argument(
        "--json",
        action="store_true",
        help="Print structured JSON instead of plain text.",
    )
    args = parser.parse_args()

    if not args.list and not args.query:
        parser.error("either provide a search query or use --list")
    if args.max is not None and args.max < 1:
        parser.error("--max must be >= 1")
    if args.snippets is not None and args.snippets < 1:
        parser.error("--snippets must be >= 1")
    if args.date:
        try:
            datetime.strptime(args.date, "%Y-%m-%d")
        except ValueError as exc:
            parser.error(f"--date must be YYYY-MM-DD: {exc}")
    return args


def use_color(args: argparse.Namespace) -> bool:
    return sys.stdout.isatty() and not args.no_color


def colorize(text: str, code: str, enabled: bool) -> str:
    if not enabled:
        return text
    return f"\033[{code}m{text}\033[0m"


def normalize_whitespace(text: str) -> str:
    return " ".join(str(text).split())


def parse_timestamp(raw: object) -> Optional[datetime]:
    if not isinstance(raw, str) or not raw:
        return None
    try:
        return datetime.fromisoformat(raw.replace("Z", "+00:00")).astimezone()
    except ValueError:
        return None


def collect_text_blocks(content: object) -> Iterable[str]:
    if isinstance(content, str):
        yield content
        return

    if isinstance(content, list):
        for block in content:
            if not isinstance(block, dict):
                continue
            if block.get("type") == "text":
                text = block.get("text")
                if isinstance(text, str) and text.strip():
                    yield text
        return

    if isinstance(content, dict) and content.get("type") == "text":
        text = content.get("text")
        if isinstance(text, str) and text.strip():
            yield text


def fallback_title(session_id: str, last_prompt: str, messages: list[MessageEntry]) -> str:
    if last_prompt:
        return truncate_plain(last_prompt, 56)
    if messages:
        return truncate_plain(messages[0].text, 56)
    return f"(untitled) {session_id[:8]}"


def truncate_plain(text: str, limit: int) -> str:
    text = normalize_whitespace(text)
    if len(text) <= limit:
        return text
    return text[: max(0, limit - 1)] + "…"


def load_session(path: Path) -> Optional[SessionEntry]:
    try:
        stat = path.stat()
    except OSError:
        return None

    project = path.parent.name
    session_id = path.stem
    fallback_time = datetime.fromtimestamp(stat.st_mtime, tz=timezone.utc).astimezone()
    updated_at = fallback_time
    title = ""
    last_prompt = ""
    messages: list[MessageEntry] = []

    try:
        with path.open("r", encoding="utf-8", errors="replace") as handle:
            for raw_line in handle:
                line = raw_line.strip()
                if not line:
                    continue
                try:
                    item = json.loads(line)
                except json.JSONDecodeError:
                    continue

                record_type = item.get("type")
                record_time = parse_timestamp(item.get("timestamp"))
                if record_time and record_time > updated_at:
                    updated_at = record_time

                if record_type == "custom-title":
                    value = normalize_whitespace(item.get("customTitle", ""))
                    if value:
                        title = value
                    continue

                if record_type == "last-prompt":
                    value = normalize_whitespace(item.get("lastPrompt", ""))
                    if value:
                        last_prompt = value
                    continue

                if record_type not in {"user", "assistant"}:
                    continue

                if item.get("isMeta"):
                    continue

                content = (item.get("message") or {}).get("content")
                text = normalize_whitespace(" ".join(collect_text_blocks(content)))
                if not text:
                    continue
                messages.append(MessageEntry(role=record_type, text=text, timestamp=record_time))
    except OSError:
        return None

    return SessionEntry(
        session_id=session_id,
        project=project,
        path=path,
        title=title or fallback_title(session_id, last_prompt, messages),
        updated_at=updated_at,
        last_prompt=last_prompt,
        messages=messages,
    )


def iter_sessions(root: Path, project_filter: Optional[str]) -> Iterable[SessionEntry]:
    if not root.exists():
        return

    project_match = project_filter.casefold() if project_filter else None
    for path in sorted(root.rglob("*.jsonl")):
        if not path.is_file():
            continue
        if project_match and project_match not in path.parent.name.casefold():
            continue
        session = load_session(path)
        if session:
            yield session


def compile_pattern(query: str) -> re.Pattern[str]:
    return re.compile(re.escape(query), re.IGNORECASE)


def build_snippet(text: str, pattern: re.Pattern[str], colors: bool) -> str:
    text = normalize_whitespace(text)
    match = pattern.search(text)
    if not match:
        return truncate_plain(text, SNIPPET_WIDTH * 2)

    start = max(0, match.start() - SNIPPET_WIDTH)
    end = min(len(text), match.end() + SNIPPET_WIDTH)
    prefix = "…" if start > 0 else ""
    suffix = "…" if end < len(text) else ""
    snippet = prefix + text[start:end] + suffix

    def repl(found: re.Match[str]) -> str:
        return colorize(found.group(0), "1;31", colors)

    return pattern.sub(repl, snippet)


def format_time(value: datetime) -> str:
    return value.astimezone().strftime("%Y-%m-%d %H:%M")


def session_to_dict(session: SessionEntry) -> dict[str, object]:
    return {
        "session_id": session.session_id,
        "project": session.project,
        "path": str(session.path),
        "title": session.title,
        "updated_at": session.updated_at.astimezone().isoformat(),
        "updated_at_display": format_time(session.updated_at),
        "last_prompt": session.last_prompt,
    }


def local_date(value: datetime) -> date:
    return value.astimezone().date()


def filter_by_date(sessions: list[SessionEntry], date_text: Optional[str]) -> list[SessionEntry]:
    if not date_text:
        return list(sessions)
    target = datetime.strptime(date_text, "%Y-%m-%d").date()
    return [session for session in sessions if local_date(session.updated_at) == target]


def print_session_header(
    session: SessionEntry, index: int, total: int, colors: bool, show_path: bool
) -> None:
    print(colorize(f"[{index}/{total}] {session.title}", "1;36", colors))
    print(f"  Time:    {format_time(session.updated_at)}")
    print(f"  Project: {session.project}")
    print(f"  Session: {session.session_id}")
    print(f"  Resume:  claude --resume {session.session_id}")
    if show_path:
        print(f"  Path:    {session.path}")


def list_session_rows(
    sessions: list[SessionEntry], query: Optional[str]
) -> list[SessionEntry]:
    if not query:
        return list(sessions)

    pattern = compile_pattern(query)
    return [
        session
        for session in sessions
        if pattern.search(session.title)
        or pattern.search(session.last_prompt)
        or pattern.search(session.session_id)
        or pattern.search(session.project)
    ]


def search_session_rows(
    sessions: list[SessionEntry], query: str, snippets_limit: int, colors: bool
) -> list[tuple[SessionEntry, list[tuple[str, str]]]]:
    pattern = compile_pattern(query)
    matches: list[tuple[SessionEntry, list[tuple[str, str]]]] = []

    for session in sessions:
        snippets: list[tuple[str, str]] = []
        seen_keys: set[tuple[str, str]] = set()

        header_sources = [
            ("title", session.title),
            ("prompt", session.last_prompt),
        ]
        for label, text in header_sources:
            if text and pattern.search(text):
                snippet = build_snippet(text, pattern, colors)
                key = (label, ANSI_RE.sub("", snippet))
                if key not in seen_keys:
                    snippets.append((label, snippet))
                    seen_keys.add(key)
                if len(snippets) >= snippets_limit:
                    break

        if len(snippets) < snippets_limit:
            for message in session.messages:
                if not pattern.search(message.text):
                    continue
                label = "user" if message.role == "user" else "assistant"
                snippet = build_snippet(message.text, pattern, colors)
                key = (label, ANSI_RE.sub("", snippet))
                if key in seen_keys:
                    continue
                snippets.append((label, snippet))
                seen_keys.add(key)
                if len(snippets) >= snippets_limit:
                    break

        if snippets:
            matches.append((session, snippets))

    matches.sort(key=lambda item: item[0].updated_at, reverse=True)
    return matches


def list_sessions(sessions: list[SessionEntry], args: argparse.Namespace, colors: bool) -> int:
    filtered = list_session_rows(sessions, args.query)
    filtered.sort(key=lambda item: item.updated_at, reverse=True)
    limited = filtered[: args.max]
    if not limited:
        if args.json:
            print("[]")
        else:
            print("No sessions found.")
        return 1

    if args.json:
        print(
            json.dumps(
                [session_to_dict(session) for session in limited],
                ensure_ascii=False,
                indent=2,
            )
        )
        return 0

    for idx, session in enumerate(limited, start=1):
        if idx > 1:
            print()
        print_session_header(session, idx, len(limited), colors, args.path)
        if session.last_prompt:
            print(f"  Prompt:  {truncate_plain(session.last_prompt, 100)}")
    return 0


def search_sessions(sessions: list[SessionEntry], args: argparse.Namespace, colors: bool) -> int:
    matches = search_session_rows(sessions, args.query, args.snippets, colors)
    limited = matches[: args.max]
    if not limited:
        if args.json:
            print("[]")
        else:
            print(f'No matches for "{args.query}".')
        return 1

    if args.json:
        payload = []
        for session, snippets in limited:
            item = session_to_dict(session)
            item["snippets"] = [
                {"label": label, "text": ANSI_RE.sub("", snippet)}
                for label, snippet in snippets
            ]
            payload.append(item)
        print(json.dumps(payload, ensure_ascii=False, indent=2))
        return 0

    for idx, (session, snippets) in enumerate(limited, start=1):
        if idx > 1:
            print()
        print_session_header(session, idx, len(limited), colors, args.path)
        for label, snippet in snippets:
            print(f"  {label:>9}: {snippet}")
    return 0


def main() -> int:
    args = parse_args()
    root = Path(args.root).expanduser()
    colors = use_color(args)
    sessions = list(iter_sessions(root, args.project))
    sessions = filter_by_date(sessions, args.date)
    sessions.sort(key=lambda item: item.updated_at, reverse=True)

    if args.list:
        return list_sessions(sessions, args, colors)
    return search_sessions(sessions, args, colors)


if __name__ == "__main__":
    sys.exit(main())
```

## 14. 一句话总结

- 现在你主要只需要记住一个命令：`cchat`
- `cchat` 同时支持弹窗和终端模式
- `csearch` 仍然存在，但主要作为内部搜索引擎
- 换电脑时，仍然要把两个脚本一起带过去
- 真正日常使用时，尽量只围绕 `cchat` 建立习惯

如果以后还想继续增强，这几个方向最值得做：

1. 给 `cchat` 做成双击可开的 `.app`
2. 给 `cchat` 增加“今天 / 昨天 / 最近 7 天”快捷入口
3. 给 `cchat` 增加导出 Markdown / CSV 的能力
